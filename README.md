## ğŸ“– Overview
**AgriTrust** is a decentralized application (DApp) designed to bring transparency and trust to the agricultural supply chain. By leveraging the **Ethereum Blockchain**, it eliminates the "black box" of food sourcing, ensuring that end consumers can verify the origin, quality, and journey of their produce.

This project was selected as a **Finalist in the Smart India Hackathon (SIH) 2025**.

## ğŸš€ Key Features
* **Immutable Record Keeping:** All transaction data (harvesting, processing, shipping) is stored on-chain using Solidity Smart Contracts.
* **End-to-End Traceability:** A QR-code based system allowing consumers to trace products back to the specific farm.
* **Tamper-Proof:** Once data is entered onto the blockchain, it cannot be altered, preventing fraud in organic or premium labeling.
* **Hybrid Architecture:** Uses **MongoDB** for heavy media storage (images/logs) and **Ethereum** for critical verification data to optimize gas costs.
